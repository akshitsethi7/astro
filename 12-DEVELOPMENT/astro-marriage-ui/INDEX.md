## 🎯 Common Questions

### Installation
**Q: What do I need to install?**
A: Node.js 18+ and npm. See [INSTALL_INSTRUCTIONS.md](./INSTALL_INSTRUCTIONS.md#prerequisites-check)

**Q: How long does installation take?**
A: 2-5 minutes. See [INSTALL_INSTRUCTIONS.md](./INSTALL_INSTRUCTIONS.md#step-by-step-installation)

**Q: What if I get errors?**
A: Check [Common Issues](./INSTALL_INSTRUCTIONS.md#common-installation-issues)

### Usage
**Q: How do I start the dev server?**
A: Run \`npm run dev\`. See [QUICKSTART.md](./QUICKSTART.md)

**Q: How do I add new content?**
A: Add markdown files to \`../marriage/\` folder

**Q: How do I customize colors?**
A: Edit \`app/globals.scss\`. See [TIPS_AND_TRICKS.md](./TIPS_AND_TRICKS.md#quick-color-changes)

### Deployment
**Q: Where can I deploy?**
A: Vercel, Netlify, or your own server. See [DEPLOYMENT.md](./DEPLOYMENT.md)

**Q: Is it free to deploy?**
A: Yes, Vercel and Netlify have free tiers

**Q: Do I need a custom domain?**
A: No, you get a free subdomain

### Development
**Q: Can I use this for other projects?**
A: Yes! It's a flexible template

**Q: How do I add new charts?**
A: See [Data Visualization Tips](./TIPS_AND_TRICKS.md#-data-visualization-tips)

**Q: Is TypeScript required?**
A: Recommended but you can use JavaScript
