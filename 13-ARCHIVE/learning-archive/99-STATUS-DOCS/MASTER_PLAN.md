3. Identify key sections
4. Note unique contributions
5. Mark important pages

#### Level 2: Deep Reading (Second Pass)
**Time**: 2-4 hours per book | **Goal**: Extract all principles

1. Read each chapter carefully
2. Take detailed notes
3. Extract all sutras/principles
4. Note all examples
5. Identify calculation methods
6. Mark cross-references

#### Level 3: Integration (Third Pass)
**Time**: 1-2 hours per book | **Goal**: Synthesize with other sources

1. Compare with other books
2. Note agreements/disagreements
3. Identify unique insights
4. Create synthesis notes
5. Plan document structure

#### Level 4: Documentation (Output)
**Time**: 2-3 hours per topic | **Goal**: Create comprehensive markdown

1. Structure content logically
2. Write clear explanations
3. Add all examples
4. Include practice exercises
5. Cross-reference sources
6. Add case studies
