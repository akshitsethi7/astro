# Astro - Complete Vedic Astrology Analysis System

A comprehensive Vedic astrology analysis and learning system with 14,069+ classical quotes from 33 books, complete chart analysis, and integrated learning materials.

## 🌟 Features

- **Complete Chart Analysis**: Personal birth chart analysis with marriage timing, career predictions, wealth analysis, and health insights
- **14,069 Classical Quotes**: Extracted and organized from 33 classical Vedic astrology texts
- **Integrated Learning System**: 80+ learning documents covering all aspects of Vedic astrology
- **PDF Extraction Tools**: 30+ Python scripts to extract content from any astrology book
- **Web UI**: Interactive Next.js application for exploring chart data
- **Professional Organization**: 20 logically organized folders with clear purposes

## 🛠️ Installation

### Prerequisites:
- Python 3.8+
- Node.js 16+ (for Web UI)

### Setup:
```bash
# Clone the repository
git clone https://github.com/yourusername/astro.git
cd astro

# Install Python dependencies
pip install -r 11-CONFIG-DATA/requirements.txt

# For Web UI (optional)
cd 12-DEVELOPMENT/astro-marriage-ui
npm install
npm run dev
```
